# CSS FLEXBOX DIO

### Proposta DIO

- Aprender na pratica a usar flexbox.

### Pré-requisitos

- IDE - De sua preferencia.

### Instalando

- Primeiro, instale o IDE corretamente.
- Em segundo lugar, baixe o projeto [Git](https://github.com/henriqss/CSS-FLEXBOX-DIO) e abra-o no ide.
